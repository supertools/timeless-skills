# Timeless Skills

Skills for AI agents to interact with [Timeless](https://timeless.day) meeting data.

## Skills

| Skill | Description |
|-------|-------------|
| [timeless-api](timeless-api/) | Query meetings, transcripts, rooms, summaries. Upload recordings. Chat with Timeless AI. Build automations on new meetings. |
| [timeless-capture](timeless-capture/) | Capture podcast episodes and YouTube videos into Timeless. Organize into rooms. |

## Shared Resources

- [api-reference.md](api-reference.md) — Full Timeless API endpoint documentation

## Setup

All skills require a `TIMELESS_ACCESS_TOKEN` env var. Get your token at [my.timeless.day/api-token](https://my.timeless.day/api-token).
